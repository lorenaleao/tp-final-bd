IMDb

Toda esta aplicação usa como base o banco de dados o IMDb (Internet Movie Database). Subconjuntos de dados da IMDb estão disponíveis para acesso por usuários para uso pessoal e não comercial. Você pode ter cópias locais desses dados e está sujeito aos nossos termos e condições. Por favor, consulte o Licenciamento Não Comercial e os direitos autorais / licença e verifique a conformidade.

Localização dos Dados

Os arquivos com os dados podem ser acessados e baixados em https://datasets.imdbws.com/. Os dados são atualizados diariamente.

Detalhes dos Dados

Cada conjunto de dados está contido em um arquivo formatado com valores separados por tabulação (.tsv), no conjunto de caracteres UTF-8. A primeira linha de cada arquivo contém cabeçalhos que descrevem o que está em cada coluna.
